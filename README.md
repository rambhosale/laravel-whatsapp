## Installation

You can install the package via composer:

```bash
composer require sawirricardo/whatsapp-php
```

## Usage

```php
$client = \Rambhosale\Whatsapp\Whatsapp::make($token, $phoneId)
    ->to('+111111111')
    ->message(TextMessageData::make('Hello world!'))
    ->send();
```
